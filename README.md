# RimaTon (Exaroton Client)

This Exaroton Client Is Designed To Works With Any Platform And Using to control server basics <br>
Examples:<br>
- Start/Stop Server<br>
- Change Ram Plans<br>
- Modify MOTD<br>
- Check Logs<br>
- View Console & Send Console Command<br>


Feature Roadmap<br>
- [ ] Logging In Via API Token <br>
- [ ] List Servers From Your accounts <br>
- [ ] Start/Stop Server <br>
- [ ] Server Panel <br>
- [ ] File Browser <br>
- [ ] Logs Viewer <br>
- [ ] Server Settings <br>
(And many more to add)
