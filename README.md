# RimaTon (Exaroton Client)

This Exaroton Client Is Designed To Works With Any Platform And Using to control server basics 
Examples:
- Start/Stop Server
- Change Ram Plans
- Modify MOTD
- Check Logs
- View Console & Send Console Command
